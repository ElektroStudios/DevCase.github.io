BroadcastSystemMessageExInfo Fields

Fields

Hdesk A desktop handle to the window specified by Hwnd.

This value is returned only if BroadcastSystemMessageEx(BroadcastSystemMessageFlags, BroadcastSystemMessageInfo, WindowMessages, IntPtr, IntPtr, BroadcastSystemMessageExInfo) specifies ReturnHDesk and Query.

Hwnd A handle to the window that denied the request.

This value is returned only if BroadcastSystemMessageEx(BroadcastSystemMessageFlags, BroadcastSystemMessageInfo, WindowMessages, IntPtr, IntPtr, BroadcastSystemMessageExInfo) specifies Query.

Luid A locally unique identifier (LUID) for the window.
SizeOfStruct The size of the structure, in bytes.

This member must be set to Marshal.SizeOf(Of BroadcastSystemMessageExInfo) before calling any function.
