KeyboardInput Fields

Fields

ExtraInfo An additional value associated with the keystroke.

Use the GetMessageExtraInfo function to obtain this information.

Flags Specifies various aspects of a keystroke.
ScanCode A hardware scan code for the key.

If Flags specifies Unicode, ScanCode specifies a Unicode character which is to be sent to the foreground application.

Time The time stamp for the event, in milliseconds.

If this parameter is 0, the system will provide its own time stamp.

VirtualKey A virtual-key code.

The code must be a value in the range 1 to 254.

If the Flags member specifies Unicode, VirtualKey must be 0.
