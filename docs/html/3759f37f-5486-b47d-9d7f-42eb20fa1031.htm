DisplayConfigModeInfo Fields

Fields

AdapterId The identifier of the adapter that this source or target mode information relates to.
DesktopImageInfo Describes information about the desktop image, only when InfoType is DesktopImage.

Supported starting in Windows 10.

Id The source or target identifier on the specified adapter that this path relates to.
InfoType A value that indicates whether the DisplayConfigModeInfo structure represents source or target mode information.

If InfoType is Target, the TargetMode field contains a valid DisplayConfigTargetMode structure describing the specified target.

If InfoType is Source, the SourceMode field contains a valid DisplayConfigSourceMode structure describing the specified source.

SourceMode Describes the specified source, only when InfoType is Source.
TargetMode Describes the specified target, only when InfoType is Target.
