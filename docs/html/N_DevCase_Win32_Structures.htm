DisplayConfigPathSourceInfo
Contains source information for a single path.

A source corresponds to a surface on which the display adapter can render pixels.

Each display adapter is capable of rendering to x number of sources. What this means is how many desktops can be rendered for extend mode. This is typically 2. For example, source 0 might be rendering pixels from 0,0 to 1024,768, and source 1 might be rendering pixels from 1025,0 to 2048, 768.
