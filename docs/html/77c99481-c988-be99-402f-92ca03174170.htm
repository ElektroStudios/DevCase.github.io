Properties

BiosUuId Gets the BIOS UUID.
CoresPerProcessor Gets the amount of cores per processor of this VM.
DisplayName Gets the display name of this VM.
Firmware Gets the firmware type of this VM. It can be: BIOS, or UEFI.
GraphicsHardwareAccelerationEnabled Gets a value that determine whether 3D graphics hardware acceleration is enabled in this VM.
GraphicsMemorySize Gets the total graphics memory size of this VM, in megabytes.
GuestOsCredential Gets or sets the username and password of the running user-account in the guest operating system of this VM.

The credential is required to perform some I/O operations with VMWare's vmrun.exe program. So you must set this credential before using vmrun.exe.

IsSharedVm Gets a value that determine whether this VM is a shared VM.
MemorySize Gets the total memory size of this VM, in megabytes.
OsVersion Gets the version of the guest operating system of this VM.
ProcessorCount Gets the amount of processors of this VM.

The resulting value is the division between TotalProcessorCores \ CoresPerProcessor.

SecureBootEnabled Gets a value that determine whether secureboot is enabled for UEFI firmware mode.
SharedFolders Gets the shared folders of this VM.
TotalProcessorCores Gets the amount of processor cores of this VM.
VmGenerationId Gets the VM-Generation ID.
VmHardwareVersion Gets the hardware version of this VM.
VmxFile Gets .vmx file of this VM.
