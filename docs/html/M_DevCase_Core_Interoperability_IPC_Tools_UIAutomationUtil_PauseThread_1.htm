Parameters

threadId
Type: SystemInt32
The thread identifier.

Return Value

Type: SafeAccessTokenHandle
The return value is an openthread-handle to the thread handle.

An openthread-handle could be used to suspend, resume or terminate a thread that the open-handle refers to.

Exceptions
ExceptionCondition
Win32Exception
Remarks
After pausing a thread, call ResumeThread(SafeAccessTokenHandle) function to resume the thread and close the openthread-handle.

To manually close an openthread-handle, call CloseHandle(IntPtr) function.

Examples
This is a code example.

Copy
Dim threadId As ProcessThread = Process.GetProcessesByName("cmd.exe").FirstOrDefault.Threads(0).Id

Dim openThreadHandle As SafeAccessTokenHandle = PauseThread(threadId)
