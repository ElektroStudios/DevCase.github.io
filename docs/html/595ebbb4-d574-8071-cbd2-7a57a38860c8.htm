UrlSlugOptions Properties

Properties

AllowedCharacterRanges Gets or sets the list of allowed character ranges.

Default value is: a-z (97 to 122) + A-Z (65 to 90) + 0-9 (48 to 57)

AllowedUnicodeCategories Gets or sets the list of allowed Unicode categories.

Default value is: UppercaseLetter, LowercaseLetter and DecimalDigitNumber.

CanEndWithSeparator Gets or sets a value indicating whether the URL slug string can end with a (Separator) separator string.
Culture Gets or sets the culture for string case conversion.
EarlyTruncate Gets or sets a value indicating whether the string is truncated before normalization.
MaximumLength Gets or sets the maximum length of a generated URL slug.

Default value is Int32.

Separator Gets or sets the character that separate words in the URL slug.

Default value is '-'.

ToLower Gets or sets a value indicating whether to lower-case the resulting string.
ToUpper Gets or sets a value indicating whether to upper-case the resulting string.
