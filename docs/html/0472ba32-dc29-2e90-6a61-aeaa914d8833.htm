BrushBack A handle to the brush to be used for the menu's background.
ContextHelpID The context help identifier.

This is the same value used in the GetMenuContextHelpId and SetMenuContextHelpId functions.

Mask The members to be retrieved or set (except for ApplyToSubmenus).

This member can be one or more of the MenuMask enumeration values.

Max The maximum height of the menu in pixels.

When the menu items exceed the space available, scroll bars are automatically used.

The default (0) is the screen height.

MenuData An application-defined value.
SizeOfStruct The size of the structure, in bytes.

This member must be set to Marshal.SizeOf(Of MenuInfo) before calling any function.

Style The menu style.
