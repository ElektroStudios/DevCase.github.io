UtilDebugNetworking Properties

Properties

IsAntiNetworkDebugProtectionEnabled Gets a value that indicates whether Anti-Network Debugging protection is enabled for the current application.

Anti-Network Debugging protection can be enabled by calling EnableAntiNetworkDebugProtection(Action) method, and disabled by calling DisableAntiNetworkDebugProtection method.
