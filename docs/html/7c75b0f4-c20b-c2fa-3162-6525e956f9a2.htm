User32DefFrameProc Method

Provides default processing for any window messages that the window procedure of a multiple-document interface (MDI) frame window does not process.

All window messages that are not explicitly processed by the window procedure must be passed to the DefFrameProc(IntPtr, IntPtr, UInt32, IntPtr, IntPtr) function, not the DefWindowProc(IntPtr, UInt32, IntPtr, IntPtr) function.

Return Value

IntPtr
The return value specifies the result of the message processing and depends on the message.

If the hWndMDIClient parameter is Zero, the return value is the same as for the DefWindowProc(IntPtr, UInt32, IntPtr, IntPtr) function.
