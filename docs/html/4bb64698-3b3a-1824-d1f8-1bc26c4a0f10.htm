UtilSteam Properties

Properties

AccountRecoveryUrl Gets the official Steam url about regaining access to a lost Steam account, the url is in the same language of the Steam client installed on the current Operating System.

Not available for all languages.

ClientCulture Gets a CultureInfo instance that represents the language of the Steam client installed on the current Operating System.
ClientFile Gets the executable file of the Steam client installed on the current Operating System.
ClientLanguage Gets the language of the Steam client installed on the current Operating System.
InstallDir Gets the installation directory of the Steam client installed on the current Operating System.
IsInstalled Determines whether Steam client is installed in the current Operating System.
RegistryInstallPath Gets the default Steam registry path.
RegistryUninstallPath Gets the default Steam Uninstall registry path.
Version Gets the version of the Steam client installed on the current Operating System.
