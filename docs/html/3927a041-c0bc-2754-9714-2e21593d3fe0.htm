UnwrappedCollectionEditorT Class

Provides a user interface that unwraps the items of a collection from the default "Value" property that is shown in a standard CollectionEditor.

Use UnwrappedCollectionEditorT if you want to show a collection whose items are types with properties that have category attributes (CategoryAttribute) applied.
