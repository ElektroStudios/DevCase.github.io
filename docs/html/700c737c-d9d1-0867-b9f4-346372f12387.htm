NativeWindowInfo Fields

Fields

AtomWindowType The window class atom.
BorderHeight The border height of the window, in pixels.
BorderWidth The border width of the window, in pixels.
Bounds The rectangle that represents the window, including its nonclient elements.
ClientRectangle The rectangle that represents the client area of the window.
CreatorVersion The Microsoft Windows version of the application that created the window.
IsWindowActive Determines whether the window is active.
SizeOfStruct The size of the structure, in bytes.

This member must be set to Marshal.SizeOf(Of NativeWindowInfo) before calling any function.

WindowExtendedStyle The extended window style.
WindowStyle The window style.
