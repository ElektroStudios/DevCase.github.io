UtilSeleniumMicrosoftLiveMailRegister Method

Automate the specified web-browser to register a new mail account at 'https://signup.live.com/'.

Note that this function use random text (gathered from UtilStringWordLists class properties) to fill the registration form.

nineKwCaptchaApiKey  String
Your API key of http://9kw.eu/ service, required to automate the captcha image solvention.

This function internally create an instance of NineKwCaptchaHelper class with the provided API key.

Return Value

A MailAccount object that represents the new registered mail account.

Returns if the function failed to register the new mail account.
