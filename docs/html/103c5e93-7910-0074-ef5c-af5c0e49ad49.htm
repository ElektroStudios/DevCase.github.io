WindowPlacement Fields

Fields

Flags Specifies flags that control the position of the minimized window and the method by which the window is restored.
MaxPosition The coordinates of the window's upper-left corner when the window is maximized.
MinPosition The coordinates of the window's upper-left corner when the window is minimized.
NormalPosition The window's coordinates when the window is in the restored position.
SizeOfStruct The size of the structure, in bytes.

This member must be set to Marshal.SizeOf(Of WindowPlacement) before calling any function.

GetWindowPlacement(IntPtr, WindowPlacement) and SetWindowPlacement(IntPtr, WindowPlacement) fail if this member is not set correctly.

WindowState The current show state of the window.
