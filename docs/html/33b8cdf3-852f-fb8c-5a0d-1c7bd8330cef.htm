FileTypeMatch Properties

Properties

FileHeader Gets the source file header.
FileType Gets the FileType that has been matched in the source file header.
PercentMatch Gets the percentage match of the matched file type on the source file header, from 0% to 100%.

This value determines the success percentage, for example a value of 100% indicates that all the file type signatures were found in the file header; a value of 50% would indicate that only half of the file type signatures were found in the file header.

SignatureMatchCount Gets the amount of signatures that were found in the source file header.
