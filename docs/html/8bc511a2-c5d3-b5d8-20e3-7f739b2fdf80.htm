WaitForIdle(Process) Causes the Process component to wait indefinitely until the user interface has been fully loaded then entered in IDLE state.

This applies only to processes with a user interface and, therefore, a message loop.

WaitForIdle(Process, Int32) Causes the Process component to wait for the specified amount of time, in milliseconds, until the user interface has been fully loaded then entered in IDLE state.

This applies only to processes with a user interface and, therefore, a message loop.
