MemoryStatusEx Fields

Fields

ExtendedVirtualAvailable Size of unreserved and uncommitted memory in the extended portion of the virtual address space of the calling process, in bytes.
MemoryLoad Number between 0 and 100 that specifies the approximate percentage of physical memory that is in use

(0 indicates no memory use and 100 indicates full memory use).

PageFileAvailable Size of available memory to commit, in bytes.

The limit is PageFileTotal.

PageFileTotal Size of the committed memory limit, in bytes.

This is physical memory plus the size of the page file, minus a small overhead.

PhysicalAvailable Size of physical memory available, in bytes.
PhysicalTotal Total size of physical memory, in bytes.
SizeOfStruct The size of the structure, in bytes.

This member must be set to Marshal.SizeOf(Of MemoryStatusEx) before calling any function.

VirtualAvailable Size of unreserved and uncommitted memory in the user mode portion of the virtual address space of the calling process, in bytes.
VirtualTotal Total size of the user mode portion of the virtual address space of the calling process, in bytes.
