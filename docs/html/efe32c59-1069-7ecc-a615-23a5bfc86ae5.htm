Properties

EnableMoveOpacity Gets or sets a value indicating whether the transparency is enabled when moving the window.
EnableResizeOpacity Gets or sets a value indicating whether the transparency is enabled when resizing the window.
Handle Gets the handle for the window that owns this WindowMovingOpacity instance.
OpacityNonStatic Gets or sets the opacity to apply to the window when the window is moving or resizing.

The valid range is from 0 to 100.

OpacityStatic Gets or sets the opacity to apply to the window when the window isn't moving, in a static position.

The valid range is from 0 to 100.

OwnerWindow Gets the window that owns this WindowMovingOpacity instance.
