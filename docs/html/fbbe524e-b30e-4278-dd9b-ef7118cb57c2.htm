Properties

FirewallEnabled Gets a value indicating whether the Windows Firewall is enabled for the current network profile.
FirewallEnabledFirewallNetworkProfile Gets a value indicating whether the Windows Firewall is enabled for the specified network profile.
FirewallServiceRunning Gets a value indicating whether the Windows Firewall service is enabled and running on the current machine.

Note that the service could be running but the Firewall itself could be disabled.
