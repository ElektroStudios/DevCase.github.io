DevModeDisplay Fields

Fields

DisplayFixedOutput For fixed-resolution display devices only, how the display presents a low-resolution mode on a higher-resolution display.

For example, if a display device's resolution is fixed at 1024 x 768 pixels but its mode is set to 640x480 pixels, the device can either display a 640x480 image somewhere in the interior of the 1024x768 screen space or stretch the 640x480 image to fill the larger screen space.

If DisplayFixedOutput is not set, this member must be zero.

DisplayOrientation For display devices only, the orientation at which images should be presented.

If DisplayOrientation is not set, this member must be Default.

Position For display devices only, a NativePoint struct that indicates the positional coordinates of the display device in reference to the desktop area.

The primary display device is always located at coordinates (0,0).
