Parameters

hwndOwner
Type: SystemIntPtr
A handle to the owner window that the client should specify if it displays a dialog box or message box.
pidlCount
Type: SystemUInt32
The number of file objects or subfolders specified in the pidls parameter.
pidls
Type: SystemIntPtr
The address of an array of pointers to ITEMIDLIST structures, each of which uniquely identifies a file object or subfolder relative to the parent folder.

Each item identifier list must contain exactly one SHITEMID structure followed by a terminating zero.

refIid
Type: SystemGuid
A reference to the IID of the interface to retrieve.

This can be any valid interface identifier that can be created for an item.

reserved (Optional)
Type: SystemIntPtr
Reserved. Must be Zero.

Return Value

Type: Object
When this method returns successfully, contains the interface pointer requested in refIid.
