DisplayConfigSourceMode Fields

Fields

Height The height, in pixels, of the source mode.
PixelFormat The pixel format of the source mode.
Position The position in the desktop coordinate space of the upper-left corner of this source surface.

The source surface that is located at (0, 0) is always the primary source surface.

Width The width, in pixels, of the source mode.
