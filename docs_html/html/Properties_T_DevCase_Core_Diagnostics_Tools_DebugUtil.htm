IsAntiNetworkDebugProtectionEnabled
Gets a value that indicates whether Anti-Network Debugging protection is enabled for the current application.

Anti-Network Debugging protection can be enabled by calling EnableAntiNetworkDebugProtection(Action) method, and disabled by calling DisableAntiNetworkDebugProtection method.

IsAntiSandboxProtectionEnabled
Gets a value that indicates whether Anti-Sandbox environment protection is enabled for the current application.

Anti-Sandbox environment protection can be enabled by calling EnableAntiSandboxProtection(Action) method, and disabled by calling DisableAntiSandboxProtection method.

VisualStudioInstances
Gets a collection of the Visual Studio instances that are running on this PC.
