Parameters

computer (Optional)
Type: SystemString
The name of the computer to be shut down.

If the value of this parameter is an empty string, the local computer is shut down.

This parameter can be an addres, for example: 127.0.0.1

timeout (Optional)
Type: SystemInt32
The number of seconds to wait before shutting down the computer.

If the value of this parameter is zero, the computer is shut down immediately.

This value is limited to MaxShutdownTimeout.

message (Optional)
Type: SystemString
The message to be displayed in the interactive shutdown dialog box.
mode (Optional)
Type: DevCase.Core.ShellShutdownMode
Indicates whether to force the shutdown.
reason (Optional)
Type: DevCase.Core.ShellShutdownReason
The reason for initiating the shutdown. By default, it is also an 'unplanned' shutdown.

If this parameter is zero, the default is an undefined shutdown that is logged as "No title for this reason could be found".

planning (Optional)
Type: DevCase.Core.ShellShutdownPlanning
